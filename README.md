# BattleRoyale (Forge 1.20.1)

마인크래프트 서버에서 배틀로얄 게임을 진행할 수 있는 **서버측 전용 Forge 1.20.1 모드**입니다.
TACZ 같은 게임플레이 모드와 함께 사용하기 위해 기존 Paper 플러그인(→ `paper-plugin` 브랜치)을
Forge 모드로 이식했습니다. 클라이언트에는 이 모드가 설치되어 있지 않아도 서버에 접속할 수 있습니다.

## 설치

1. Forge 1.20.1 (47.4.0 이상) 서버를 준비합니다.
2. [Releases](../../releases)에서 `battleroyale-*.jar`를 받아 서버의 `mods/` 폴더에 넣습니다.
3. 서버를 한 번 실행하면 `config/battleroyale-server.toml` 설정 파일이 생성됩니다.

## 명령어

### 팀 관리 명령어 (`/brteam`) — OP 전용

*   **`/brteam create <팀이름>`**: 새로운 커스텀 팀을 생성합니다.
*   **`/brteam add <팀이름> <플레이어>`**: 지정된 플레이어를 커스텀 팀에 추가합니다.
*   **`/brteam remove <팀이름> <플레이어>`**: 지정된 플레이어를 커스텀 팀에서 제거합니다.
*   **`/brteam delete <팀이름>`**: 커스텀 팀을 삭제합니다.
*   **`/brteam list`**: 현재 생성된 모든 커스텀 팀과 팀원 목록을 보여줍니다.

### 게임 시작 명령어 (`/br`) — OP 전용

*   **`/br startdefault <팀_개수>`**: 온라인 플레이어를 자동으로 팀으로 나누어 게임을 시작합니다.
*   **`/br startim <팀_개수>`**: 미리 정의된(또는 수동 설정된) 팀으로 게임을 시작합니다.

### 유틸리티 명령어

*   **`/chd`, `/총`**: 총기 작업대(TACZ 설치 시)를 지급합니다.
*   **`/기본템설정`** (OP 전용): 게임 시작 시 지급될 기본 아이템을 설정하는 GUI를 엽니다.
*   **`/기본템`, `/rlqhsxpa`**: 설정된 기본 아이템을 지급합니다.
*   **`/teamtest`** (OP 전용): 플레이어를 2개 팀으로 나누는 테스트를 실행합니다.
*   **`/팀가르기 <팀_개수>`** (OP 전용): 온라인 플레이어(관전자 제외)를 지정된 수의 팀으로 나눕니다.
*   **`/팀참여 <팀_번호>`** (OP 전용): 지정된 팀 번호에 참여합니다.
*   **`/suplytest`** (OP 전용): 현재 자기장 내 무작위 위치에 보급품을 소환합니다.
*   **`/밥`**: 빵 64개를 지급합니다.
*   **`/강제종료`** (OP 전용): 서버를 종료합니다.
*   **`/top`, `/탑`**: 현재 X, Z 좌표에서 가장 높은 블록 위로 순간이동합니다.

## 게임 시스템

### 팀 시스템
자동/커스텀 팀 분할, 스코어보드 팀 기반 전멸 판정, 팀별 스폰, 게임 시작 시 인벤토리 초기화·체력/레벨
설정·야간투시/저항 버프 지급을 지원합니다.

### 자기장 시스템 (월드보더)
`config/battleroyale-server.toml`의 `border` 항목으로 단계별 크기·대기시간·축소 속도를 설정할 수 있으며,
보스바와 액션바로 자기장 정보(크기, 다음 중심, 축소까지 남은 시간)를 표시합니다.

> **참고**: Paper 전용 API였던 "플레이어별 가짜 나침반 타겟"은 클라이언트 모드 없이는 바닐라 클라이언트에서
> 구현할 수 없어 나침반 바늘이 자기장 중심을 가리키지는 않습니다. 나침반을 들고 있을 때 보스바에 거리 정보는
> 그대로 표시됩니다.

### 기절/소생 시스템
치명적인 피해를 받으면 사망 대신 기절 상태(소생 가능)로 전환됩니다. 팀원이 근처에서 웅크리고 있으면
일정 시간 후 소생시킬 수 있고, 상대팀이 공격하면 즉시 처형됩니다. 기절 중에는 상호작용·아이템 드롭·
인벤토리 변경이 차단됩니다.

### 보급품 시스템
설정된 간격(분 단위)으로 자기장 내 무작위 위치에 보급 상자가 소환됩니다.

### 유틸리티
광물(철/금/구리/청금석/레드스톤/다이아몬드/에메랄드/석탄)과 모래의 커스텀 드롭, 특정 아이템(인챈트 황금
사과, 엔드 수정, 리스폰 앵커, 불사의 토템, 물약, 화살, 침대류) 제작 제한, 전투 외 피해(낙하·용암 등) 절반
감소를 제공합니다.

## 설정 (`config/battleroyale-server.toml`)

```toml
[game]
    max_health = 40.0
    damage_multiplier = 1.0

[border]
    sizes = [2500.0, 2000.0, 1500.0, 1000.0, 500.0, 250.0, 100.0, 50.0]
    countdown_times = [300, 240, 180, 120, 90, 60, 30]
    speed = [1.0, 1.5, 2.0, 2.5, 3.0, 3.5, 4.0]

[supply_drop]
    interval_minutes = 4
    items = ["diamond_chestplate:1:protection=4", "ender_pearl:4-8", "tnt:5-10"]

default_items = ["diamond_sword:1:sharpness=5,unbreaking=3", "cooked_beef:32"]
```

아이템 문자열 형식은 `아이템ID:수량(또는 최소-최대):인챈트=레벨,인챈트=레벨` 입니다 (예: `bow:1:power=4,infinity=1`).

## 빌드

```
./gradlew build
```

Java 17 툴체인이 필요합니다. 결과 jar는 `build/libs/`에 생성됩니다.

남아 있습니다.
